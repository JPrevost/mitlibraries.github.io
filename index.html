Making Changes to this Documentation¶

Changes to this documentation should follow the workflow outlined in GitHub Workflows. Make sure you have Sphinx installed. The source documentation is in the docs/ directory.

Create a local feature branch for your changes:

$ git clone https://github.com/MITLibraries/mitlibraries.github.io.git
$ cd MITLibraries
$ git checkout -b <feature-branch>

Make your changes to the source documentation and then build the HTML docs:

$ cd docs && make html

Commit the changes to your feature branch and push them to the remote repository:

$ git commit -a
$ git push origin <feature-branch>
